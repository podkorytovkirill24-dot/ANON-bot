# Бот анонимных сообщений

Telegram-бот для анонимной отправки сообщений в канал:
- пользователь отправляет текст/фото/видео/аудио/файл в личку боту
- админ получает заявку с кнопками `Удалить` / `Опубликовать`
- в канале публикация выходит с заголовком `Анонимное сообщение`
- для текстовых сообщений используется формат с рамкой

## Возможности
- Поддержка большинства типов сообщений Telegram
- Анонимная публикация в канал без указания отправителя
- CSV-выгрузка заявок для админа (`/csv`)
- База SQLite (`queue.db`) со статусами: `pending`, `deleted`, `published`

## Установка
1. Установите Python 3.10+.
2. Создайте виртуальное окружение и установите зависимости:
   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   pip install -r requirements.txt
   ```
3. Создайте `.env` на основе `.env.example` и заполните:
   - `BOT_TOKEN` - токен из BotFather
   - `ADMIN_CHAT_IDS` - Telegram ID админов через запятую (пример: `111111111,222222222`)
   - `CHANNEL_ID` - ID канала (для приватного обычно `-100...`)
   - `CHANNEL_NAME` - как бот называет канал в ответах пользователям (например, `Лицей`)
4. Добавьте бота админом в канал с правом публикации.
5. Запустите:
   ```powershell
   python main.py
   ```

## Команды
- `/start` - приветствие
- `/csv` - скачать CSV со всеми заявками (только админ)

## Примечания
- Пользователь не видит информацию про внутреннюю модерацию.
- Если публикация не работает, проверьте права бота в канале и корректность `CHANNEL_ID`.
- Для совместимости можно оставить старый `ADMIN_CHAT_ID`, если админ только один.
